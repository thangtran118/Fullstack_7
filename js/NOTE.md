### Regex

##

1. []: tìm kí tự trong []
2. [^]: tìm kí tự không trong []
3. {a, b}: min: a, max: b
4.  * : 0 hoặc nhiều
    + : 1 hoặc nhiều
    ? : 0 hoặc 1
5.  ^: bắt đầu chuỗi
    $: kết thúc chuỗi
6. 
    + \w: [a-zA-Z0-9]
    + \W: [^a-zA-Z0-9]
    + \d: [0-9]
    + \D: [^0-9]
    + .: mọi kí tự
